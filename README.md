# AI行业动态追踪系统

一个现代化的AI新闻聚合网站，自动抓取和展示AI行业的最新动态。

## 功能特性

- 📰 **真实AI新闻**：从量子位、雷锋网、36氪、Solidot等权威媒体抓取
- ⭐ **收藏功能**：收藏感兴趣的文章，随时回顾
- 🔍 **智能搜索**：支持按标题、摘要、标签搜索
- 📱 **响应式设计**：支持桌面和移动设备
- 🌙 **深色主题**：护眼深色界面设计
- ⏰ **定时更新**：每天早上8点自动抓取最新新闻
- 📄 **分页浏览**：每页20条，流畅浏览大量文章

## 数据源

当前支持以下权威中文AI/科技媒体：

| 来源 | 描述 | 状态 |
|------|------|------|
| **量子位** | 以"有趣有料"著称的AI新媒体 | ✅ 可用 |
| **雷锋网** | 综合性科技媒体的AI垂直频道 | ✅ 可用 |
| **36氪** | 科技创业媒体 | ✅ 可用 |
| **Solidot** | 科技资讯聚合 | ✅ 可用 |

> 机器之心、新智元等RSS源暂时不可用，已禁用

## 使用方法

### 1. 安装依赖

```bash
npm install
```

### 2. 手动抓取新闻

```bash
# 抓取所有启用的新闻源
node crawler.js

# 测试单个数据源
node crawler.js --test 量子位
node crawler.js --test 雷锋网

# 查看可用数据源
node crawler.js --list
```

### 3. 启动应用

```bash
node server.js
```

打开浏览器访问 `http://localhost:3000`

### 4. 自动定时抓取

系统已配置每天早上8点自动抓取新闻的定时任务。

## 数据存储

- 抓取的文章：`data/articles.json`
- 收藏列表：浏览器localStorage
- 订阅源配置：浏览器localStorage

## 界面说明

### 左侧菜单
- **今日动态**：展示今天抓取的文章
- **我的收藏**：展示收藏的文章
- **全部文章**：展示所有历史文章
- **订阅源**：管理新闻来源

### 文章列表
- 横向列表布局，信息紧凑
- 显示来源、标题、摘要、时间、作者、标签
- 点击⭐收藏，点击卡片查看详情

### 分页导航
- 每页显示20条文章
- 支持首页/末页快速跳转
- 显示文章总数和当前页码

## 技术栈

- 前端：纯HTML/CSS/JavaScript
- 后端：Node.js
- RSS解析：rss-parser
- 数据存储：JSON文件 + localStorage

## 扩展真实数据源

### 添加新的RSS源

编辑 `crawler.js`，在 `NEWS_SOURCES` 数组中添加：

```javascript
{
    name: '新数据源名称',
    url: 'https://example.com/feed',
    enabled: true,
    description: '数据源描述'
}
```

### RSS源要求

- 必须是标准的RSS 2.0或Atom格式
- 包含 title、link、pubDate 字段
- 可公开访问（无需登录）

## 定时任务管理

查看定时任务：
```
/claude cron list
```

取消定时任务：
```
/claude cron delete <任务ID>
```
